## 本地部署与运行指南 

### 环境要求 
*   **Python**: 3.10 或更高版本
*   **Node.js**: 18 或更高版本
*   **OpenAI API Key** 或 **Google Gemini API Key**

先创建并激活一个新的python虚拟环境

### 步骤 1: 后端启动 (Backend Setup)

1.  进入后端目录：
    ```bash
    cd backend
    ```
    
3.  安装依赖：
    ```bash
    pip install -r requirements.txt
    ```
    
5.  启动 FastAPI 服务：
    ```bash
    uvicorn main:app --reload --port 8000
    ```
    *服务将在 http://localhost:8000 启动*

### 步骤 2: 前端启动 (Frontend Setup)

1.  进入项目根目录：
    ```bash
    cd frontend
    npm install	# 应该不用，上传的时候包括了，npm run dev失败的话试试
    ```
2.  启动开发服务器：
    ```bash
    npm run dev
    ```
3.  浏览器访问控制台显示的地址 (通常是 http://localhost:5173)。

### 步骤 3: 使用流程 (Usage)

1.  **注册/登录**: 在登录页随便输入邮箱密码（注册模式），系统会在本地 SQLite (`sql_app.db`) 创建用户。
2.  **配置 Key**: 点击左下角“设置”，在“自定义模型配置”中填入您的 OpenAI API Key  和  Base URL。
3.  **上传数据库**: 点击输入框左侧的数据库图标，上传数据库文件（.sqlite）。
4.  **上传知识库 (可选)**: 点击侧边栏的“知识库管理”，上传文档。
